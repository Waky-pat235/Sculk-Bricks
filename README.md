# Sculk Bricks
<img width="1914" height="1001" alt="sculk_bricks_in_world" src="https://github.com/user-attachments/assets/8985c9cb-7ee1-4597-8f13-81865f27f878" />

<br>Decorate your base with eerily-captivating sculk bricks in a variety of colors! Sculk can be crafted into bricks and colored with any dye (but only dyes). Once dyed, sculk and sculk bricks can be recolored in any combination.<br>
